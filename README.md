# AI Python Code Agent

An LLM-powered command-line tool that can read, modify, and execute Python code using the Gemini API.

This project was built in Python as part of the **"Build an AI Agent in Python"** guided project on Boot.dev. The tool is designed to assist with Python development tasks by interacting with code files directly through a command-line interface.

## Features

* Read existing Python files
* Update and rewrite Python code
* Execute Python scripts from the command line
* Use the Gemini API to generate and modify code

## How It Works

The program acts as an AI-powered coding assistant in the terminal. It accepts user input, sends requests to the Gemini API, and performs actions on Python files based on the response.

Depending on the prompt, the agent can:

* inspect Python code
* make changes to existing files
* generate new code
* run Python scripts

This creates a simple but practical example of how large language models can be used in software development tools.

## Running the Project

1. Clone the repository
2. Install dependencies
3. Add your Gemini API key
4. Run the CLI program
